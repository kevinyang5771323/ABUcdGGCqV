# 蘑菇百科系统简介 python127

## 项目概述

蘑菇百科系统是基于Spring Boot的B/S结构应用，结合Vue、ElementUI、MySQL和Redis等技术栈构建。系统提供用户友好的蘑菇信息交流平台，并通过python大模型实现对蘑菇的实时识别功能。

## 技术栈

- 后端：Spring Boot
- 前端：Vue + ElementUI
- 数据库：MySQL
- 缓存：Redis

## 功能模块

### 系统角色

- 普通用户
- 管理员

### 普通用户功能

1. 注册、登录、退出蘑菇论坛
2. 个人信息修改
   - 修改登录密码
3. 蘑菇论坛首页
   - 轮播图
   - 热门蘑菇帖子
   - 最新蘑菇帖子
4. 蘑菇百科
   - 通过蘑菇分类标签过滤菜谱
5. 蘑菇论坛
   - 搜索论坛
   - 发布文章
   - 查看论坛榜单
   - 查看论坛详情
6. 我的收藏
   - 查看收藏的蘑菇
   - 取消收藏
   - 搜索
7. 论坛详情操作
   - 点赞论坛
   - 评论论坛
   - 回复论坛
   - 分享论坛
   - 收藏论坛

### 管理员功能

1. 主页
   - 用户数量统计
   - 论坛文章数量统计
   - 各季度用户统计
2. 轮播图管理
   - 设置系统首页轮播图
   - 搜索、新增、删除、编辑轮播图
3. 蘑菇帖子管理
   - 查看所有帖子
   - 搜索、编辑、删除、新增和查看帖子内容
4. 评论管理
   - 查看所有评论
   - 搜索、编辑和删除评论
5. 购物车管理
   - 查看所有收藏
   - 搜索、编辑和删除收藏
6. 蘑菇分类管理
   - 查看帖子的蘑菇分类
   - 搜索、编辑、新增和删除帖子分类
7. 系统管理
   - 用户管理
   - 角色管理
   - 菜单管理
   - 文件管理
8. 用户管理
   - 多条件搜索用户
   - 新增、编辑、删除用户
   - 导入和导出用户
9. 角色管理
   - 新增用户角色
   - 分配角色菜单
   - 编辑角色
   - 删除角色
   - 搜索角色
10. 菜单管理
    - 新增菜单
    - 编辑菜单
    - 删除菜单
    - 搜索菜单
11. 文件管理
    - 系统图片管理
    - 上传文件
    - 下载文件
    - 预览文件
    - 删除文件
    - 启用/停用文件

## 核心亮点
